# 3songs-api


## Tech

- Node
- Mongodb
- Mongoose
  
## Installation
 
1. clone the repo
   ```sh
   git clone https://github.com/Yonasketema/3songs-api.git
   ```
2. install packages
   ```sh
   npm i
   ```
3. Set up your .env file

   Duplicate `.env.example` to `.env`.

    ```bash
    cp .env.example .env
    ```
   
4. start client

   ```sh
   npm run dev
   ```

## Seeding Database

   ```sh
   npm run seed:db
   ```

## docker Installation   
  ```bash
  docker-compose up -d
  ```
